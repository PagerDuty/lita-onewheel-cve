Puts out links to CVEs when they're posted in chat.

Installation
----
Add lita-onewheel-cve to your Lita instance's Gemfile:

``gem 'lita-onewheel-cve'``

Usage
----
When someone mentions 'CVE-1234-1234' it'll post a link to mitre.org.

That's all, folks!
